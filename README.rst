TDDBC for Python nose
=====================

これは、TDDBCのPythonista向け `nose`_ プロジェクトです

.. _nose:https://pypi.python.org/pypi/nose/

動作環境
--------

* Python 2.7
* Python 3.2
* Python 3.3

セットアップ
------------

.. code:: sh

   pip install -r requirements.txt
   python setup.py nosetests

を実行し、以下のように出力されます
